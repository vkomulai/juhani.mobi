# Juhani.mobi Feature Summary

## Shopping List
- Add items to list via voice
- Mark items as collected with strikethrough
- Remove individual items with trash icon
- Clear entire list with one tap
- Reorder items via long-press drag-and-drop
- Duplicate items ignored automatically

## Voice Input
- Finnish speech recognition via browser API
- English speech recognition supported
- Combines spoken numbers and units with item names
- Combines adjectives with following nouns
- Corrects known speech recognition errors

## Smart Sorting
- Auto-sort items by grocery store aisle order
- Nine Finnish market categories (produce to household)
- Fuzzy matching assigns items to categories
- Unknown items placed at end of list
- Toggle auto-sort on or off in settings

## Sharing & Recipes
- Share shopping list via Android share menu
- Recipients open shared list via unique URL
- Import recipe ingredients from shared URLs
- Scraped recipes cached on backend

## Offline & Installation
- Installable as standalone app on Android
- Works offline with cached content
- Offline indicator shown when disconnected
- Auto-updates when new version available

## Settings
- Slide-in hamburger menu
- Switch between Finnish and English
- Toggle automatic category sorting
- Shows app version and build date
